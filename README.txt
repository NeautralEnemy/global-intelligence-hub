Global Intelligence Hub V1.5

How to use
- Unzip the folder
- Double-click `global-intelligence-hub-v1.5.html`

What V1.5 adds on top of Version 1
- Flag beside the country name
- Calling code
- Capital
- ISO 2 / ISO 3 codes
- Area
- Timezone
- Currencies
- Languages
- Neighbouring countries
- Better right-hand info panel
- Notes and watchlist saved locally

Important note
- This keeps the simpler Version 1 structure.
- The country dataset is embedded in the HTML.
- The optional live intelligence feed may still be blocked by file-mode browser restrictions, but the main app should still open and work.

- Rival Nations section in the country profile overview

- Intel feed requests are now limited to articles from the last 6 hours when GDELT responds.

- Article cards now show publish date and time when available

- Overview tab now includes Daily Oil Use (est.) in barrels per day
- Major countries use bundled starter values; others use a simple regional population-based estimate

- Added Daily Oil Production metric
- Enables quick comparison of oil consumption vs production per country

- Added military metrics: budget, army size, aircraft, warships

- Added nuclear capability, alliances, overseas bases, defence GDP %, active conflict indicator
